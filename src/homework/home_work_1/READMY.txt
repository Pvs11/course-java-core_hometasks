Задание Producer-Queue-Consumer
ShopQueue хранит очередь элеметов которые пользователь вводит с консоли.
Producer класс запускает поток который предлагает пользоватею ввести данные и записывает эти данные в ShopQueue
Consumer класс вытягивает из очереди число и засыпает на количесвто секунд которое равно этому числу, если чисел нет то просто спит 1 сек
Папка goods содержит запись результата работы.
Resources - для логирования
Пакет exception для обработки некорректного ввода
Запускается из ShopDemo

Задание Censor
Все классы находятся в пакете censor
FileQueue хранит очередь файлов папке, которую указывает пользователь
FileQueueFiller заполняет эту очередь
FileSearchAndMerge - берет файл из очереди, считывает его и записывает дописывает в новый общий файл
FileContentModification - принимает файл со словами которые надо удалить из общего файла и создает новый модифицированный файл.
Запускается из FileSearchAndMergeDemo
